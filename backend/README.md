# Getting Started with contest apis

You can create contest, verify contest, register participant

### Install 

##### Step 1

Download dependency

---
```
npm i
```

##### Step 2

Check database is there contests and participant_registatios table or not
if not found thos table then

---
```
npm run dbsetup
```
##### Step 3

For development 

---
```
npm run dev
```
For production
```
npm start
```
